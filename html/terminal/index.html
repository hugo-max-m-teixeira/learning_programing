Como usar o terminal bash (Linux)

Outros nomes para a linha de comando:

  • Bash
  • Terminal
  • Console
  • Prompt

Por que usar isso?

  • Por lá você tem acesso a recursos mais avançados do sistema
  • Realiza certas tarefas de maneira mais rápida
  • Exemplo: versionamento com git, criação de diretórios(pastas), de arquivos rápidos...

Como entrar no terminal

  • Se você estiver usando Linux: Pode pressionar a tecla super (bandeira do Windows) e pesquisar por terminal ou simplesmente usar o atalho de teclado Crtr + Alt + T
  • Se você estiver usando Windows: Terá que instalar algum aplicativo para que os comandos que veremos abaixo funcionem, sugiro fortemente que você instale o Git (usaremos ele mais tarde para fazer o versionamento de nossos códigos). Após instalar, pressione a tecla super (bandeirinha do Windos) e pesquise por Git Bash, esse será o terminal que usaremos.

Alguns comandos...

  • pwd - (print working directory) - mostra onde você está(diretório atual)
  • cd - (change directory) - leva-te para outro lugar (muda de diretório)
  • ls - (list) - lista os diretórios dentro do diretório atual
  • clear - (clear mesmo kkkkk) - limpa o terminal
  • touch texto.txt - cria um aquivo chamado texto no formato/extensão .txt
  • cat texto.txt - mostra o conteúdo do arquivo texto.txt
  • rm texto.txt - remove/apaga/deleta o arquivo texto.txt
  • rm -r minha_pasta - Apaga a pasta minha_pasta e os arquivos que estiverem dentro dela (apaga recursivamente)
  • nano arquivo.txt - nano é um editor de texto que já vem instalado por padrão, aqui estamos editando o arquivo arquivo.txt

E bola para frente...
