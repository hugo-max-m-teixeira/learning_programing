Versionamento com Git

Já pensou se fosse possível "voltar no tempo"?

Com o Git isso é possível :)

Mas como usar essa coisa?

  1. Crie o repositório de trabalho(onde ficarão os arquivos, programas) caso ele ainda não exista
  2. Abra o terminal e, dentro do repositório de trabalho, inicie o Git com o comando "git init"
  3. Agora é necessário informar para o Git quais arquivos queremos para ele monitorar. Para isso use: git add arquivo-tal.txt , substituindo "arquivo-tal.txt" pelo nome do arquivo desejado
  4. Para dar commit (criar um ponto na história, um porto seguro para vaoltar em caso de pânico) digite: git commit -m "meu-commit" substitua "meu-commit" pela mensagem que você quer deixar para referenciar(para lembrar, caracterizar) esse commit

OK, legal. Mas agora quero enviar meus commits para o GitHub. Como fazer isso?

  1. git remote add origin "link do seu repositório remoto"
  2. git push -u origin master
